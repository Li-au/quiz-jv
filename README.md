# 🎮 Quiz Jeux Vidéo

Teste tes connaissances en jeux vidéo — reconnais un jeu uniquement depuis sa description !

**👉 [Jouer en ligne](https://li-au.github.io/quiz-jv/)**

## Comment jouer

Lis l'indice et choisis le bon jeu parmi 4 propositions. Plus tu réponds vite, plus tu marques de points. Enchaîne les bonnes réponses pour activer un multiplicateur combo.

## Difficultés

| Niveau | Temps | Jeux |
|--------|-------|------|
| Facile | 30 sec | Classiques incontournables + mobile |
| Moyen | 20 sec | Titres moins évidents |
| Difficile | 12 sec | Pour les vrais gamers |

## Fonctionnalités

- 39 questions (PC, console et mobile)
- Système de combo jusqu'à x3
- Bonus de temps selon la rapidité
- Écran de résultats détaillé

## Technologies

HTML · CSS · JavaScript (vanilla)
